## 프로젝트 소개: FromOfYou

**FromOfYou**는 Next.js (TypeScript) 기반으로 개발 중인 아트·디자인 큐레이션 플랫폼입니다.  
전시, 작가, 작품, 큐레이션 콘텐츠를 효율적으로 관리하고 사용자에게 제공합니다.

### 프로젝트 목표
FromOfYou는 다음을 목표로 합니다:
- 예술, 디자인, 전시 관련 큐레이션 콘텐츠 제공
- 작가, 작품, 큐레이션 시리즈의 체계적 관리
- 반응형, SEO 최적화된 웹 애플리케이션 구축

### 기술 스택
- **프레임워크:** Next.js 14, React 18, TypeScript
- **스타일링:** (예정) Tailwind CSS / styled-components
- **데이터베이스:** Supabase (인증 및 데이터 관리)
- **배포:** Vercel

### 주요 기능 (예정)
- 관리자용 대시보드: 전시, 작가, 작품, 큐레이션 관리
- 공개 페이지: SEO 최적화 및 반응형 UI
- Supabase 기반 이메일/소셜 로그인 기능
